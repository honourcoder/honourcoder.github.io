swift常用语法

Swift基础

这是我的个人博客，欢迎大家常来围观。网址是：www.coderqi.com. 本系列博客是本人作为一个IOS入门者学习swift的总结,如有错误,欢迎指正.

基础语法

语言是共通的,在编程思想层次上来说,各个语言可以说是基本相同,只是在用到的时候具体的语法可能会有所不同.那么想要掌握swift,对其语法的熟悉是第一步.

常量和变量

let(var) 常量名 : 类型  =  表达式
let用于声明常量 var用于声明变量 注意:swift允许使用任何Unicode字符作为变量和常量的名字,包括那些不属于ASCII编码的字符(不过貌似然并卵).
swift中分号的书写是可选的了,默认一行结束就是一个语句的结束,但是如果你想写上当然是最好的. #####字符串 swift对字符串进行了显著的加强,前面也提到了.主要有以下几个改进的地方.
1 @前缀省略掉了
2 占位符直接写在了字符串的相应的位置中,示例代码如下: 
let year = 2015
let string = "今年是\(year),明年是\(year+1)"

3 字符串比较
swift中对字符串比较做了代码优化,只需像int类型一样直接双等号进行判断即可.

4 字符串拼接
如上篇所言,swift的字符串可以直接通过加号实现两个字符串的拼接.

类型转换

在swift中相乘和相除的两个操作数必须是一致的,在OC和C中的类型隐式转换被取消了,不同类型的操作数之间进行操作必须进行类型强制转换

区间

swift中对区间的定义进行了改进

var range = 1..<5; 上面的代码等同于="" var="" range="Range(start:" 1,="" end:="" 5);<="" code="">
上述代码创建的Range是1到5,不包括5.是半闭区间
全闭区间 
var range = 1...5;
创建的是1到5,包含5.

元组

元组是swift中新引入的一种数据类型,很像swift中的字典,但是,元组中存放的值可以是任意类型,并不要求是相同类型
1 元组的声明: 
var pek = ("name","coder_qi");
或者
var pek = (key:"name",value:"coder_qi");
该元组将两个String类型的值组合起来表示一组属性和他的值
2 元组的调用
元组的调用通过".元素所在的索引"来访问对应的成员 
println("属性是:\(pek.0),对应的值是:\(pek.1)");
或者
println("属性是:\(pek.key),对应的值是:\(pek.value)");
3 元组的分解
可以通过分解将一个元组分解成多个变量 
let (name_key , name_value) = pek;
println("name是\(name_value)");

switch

相比于OC,swift的switch支持的匹配类型新增了许多,有整型,浮点,区间,字符串,元组,还可以合并分支选项.还有一个很大的改变是,break不用再显式的去标注了,swift会自动判断当前分支是否执行结束,执行结束之后自动跳出整个分支

var i = 10;
switch i%9{
case 1,2,3:
    println(“1”);
case 4,5,6:
    println(“4”);
case 7,8,9:
    println(“7”);
default:
}

还有一个很重要的变革是支持where子句

case let (x,y) where abs(x)==abs(y):
    println(“(x),(y)位于对角线上”);

有点sql语句的味道了,第一感觉就是条件的控制更灵活了,能够控制到更细.
